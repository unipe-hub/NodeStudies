## Sobre o projeto

O **NodeStudies** nasce com a proposta de ser um **nó de estudos** entre alunos da **UNIPÊ**: um repositório comunitário para concentrar conteúdos acadêmicos de forma organizada e acessível.

Aqui, qualquer estudante pode contribuir com informações úteis para a jornada universitária, como:

- conteúdos por disciplina;
- informações sobre professores;
- dicas de estudo;
- listas de exercícios;
- provas anteriores;
- links úteis e materiais complementares.

## Objetivo

Criar uma base de conhecimento aberta e viva para a comunidade acadêmica da **UNIPÊ**, feita por estudantes para estudantes, incentivando:

- colaboração entre turmas;
- troca de experiências reais sobre as disciplinas;
- democratização do acesso a materiais de apoio;
- fortalecimento da comunidade acadêmica.

## Como funciona?

Cada disciplina terá um diretório próprio dentro do repositório. A ideia é padronizar a organização para que qualquer pessoa encontre rapidamente o que precisa.

Exemplo de estrutura por disciplina:

```text
disciplinas/
  estrutura-de-dados/
    ementa.md
    resumos/
    implementacoes/
    professores.md
    avaliacoes-e-exercicios/
    dificuldadesComuns.md
    linksUteis.md
```

Dentro de cada disciplina, os conteúdos ficam divididos assim:

| Arquivo | Finalidade |
|---|---|
| `ementa.md` | Conteúdo programático oficial da disciplina, com os tópicos que serão abordados ao longo do período. |
| `resumos/` | Resumos por assunto da disciplina, organizados em arquivos curtos e objetivos para facilitar revisão. |
| `implementacoes/` | Pasta opcional para disciplinas práticas (como Estruturas de Dados), com implementações em diferentes linguagens e explicações de uso. Exemplo: a mesma estrutura em C, Java e Python, com comentários e contexto. |
| `professores.md` | Informações sobre professores da disciplina, como método de avaliação, estilo de aula e observações úteis para os alunos. |
| `avaliacoes-e-exercicios/` | Pasta única para provas, simulados e listas de exercícios, podendo ser organizada por período, unidade ou tema. |
| `dificuldadesComuns.md` | Principais dúvidas e obstáculos enfrentados pelos alunos, com observações úteis de quem já cursou. |
| `linksUteis.md` | Curadoria de links relevantes: documentação, videoaulas, livros, simuladores e outros recursos de apoio. |


## Regras

- Respeite o código de conduta.
- Proibido falar mal dos professores. Evite ao máximo comentar opiniões de qualquer natureza sobre professores.
- Proibido publicar soluções de atividades avaliativas das disciplinas que se repetem todos os períodos.
