# Implementacoes

Esta pasta concentra as implementacoes praticas das estruturas de dados estudadas na disciplina.

## Organizacao recomendada

As implementacoes ficam agrupadas por assunto, e dentro de cada assunto por linguagem.

```text
implementacoes/
  listas-sequenciais/
    README.md
    c/
      lista_sequencial_estatica.c
    python/
    java/
    golang/
  listas-encadeadas/
    README.md
    c/
      lista_encadeada_simples.c
    go/
      lista_encadeada.go
```

## Regra de uso

- `README.md` do assunto: explica quais arquivos existem e por onde comecar.
- `c/`: linguagem principal da disciplina.
- outras linguagens: servem como comparacao e extensao.
