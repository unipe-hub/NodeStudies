## Ementa

Listas lineares: pilhas, filas e filas duplas. Alocação sequencial e encadeada. Listas circulares e duplamente encadeadas. Matrizes e listas ortogonais. Árvores: nomenclatura e representação. Algoritmos de manipulação e percurso em árvores binárias. Árvores de busca binárias. Árvores B. Árvores balanceadas. Exemplos de aplicação das diversas estruturas de dados.

## Objetivo Geral

Apropriar-se dos conhecimentos teórico-práticos de abstração de dados associados às estruturas de dados mais comuns em algoritmos computacionais.

## Objetivos Específicos

- Projetar soluções para manipulação de dados a partir de estruturas de dados.
- Construir algoritmos computacionais para solução de problemas reais utilizando as estruturas projetadas.
- Analisar e implementar técnicas avançadas de algoritmos para manipulação de dados em linguagem de programação C, Java ou Python.

## Competências e Resultados de Aprendizagem

- **Conhecimentos:** Modelagem de dados com abstração de dados, estruturas lineares e não lineares, e implementação de algoritmos em linguagem de alto nível.
- **Habilidades:** Modelar soluções para problemas reais, selecionar estruturas de dados adequadas, projetar algoritmos, testar implementações e desenvolver programas em C, Java ou Python.
- **Atitudes:** Trabalho sistematizado, raciocínio lógico e abstrato, discernimento na seleção de técnicas e ferramentas, documentação adequada e desenvolvimento profissional contínuo.
