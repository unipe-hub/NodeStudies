# Estrutura de Dados I

Este diretório reúne materiais de apoio da disciplina de **Estrutura de Dados I** do curso de **Ciência da Computação** no **UNIPÊ**.

## Visão geral

- Plano de ensino com ementa, objetivos e conteúdo programático da disciplina.
- Resumo de revisão com os fundamentos necessários antes de avançar para listas, pilhas, filas e árvores.
- Curadoria inicial de links úteis para complementar o estudo.

## Estrutura com propósito

## Conteúdos centrais da disciplina

- Listas lineares com implementação sequencial e encadeada.
- Listas ordenadas, pilhas, filas lineares, filas circulares e deques.
- Técnicas de encadeamento, recursividade, matrizes esparsas e listas ortogonais.
- Árvores, percursos, árvores binárias, árvores binárias de busca, árvores B e árvores balanceadas.

## Observações

- A disciplina exige base sólida em lógica, ponteiros e uso de memória, especialmente em `C`.
- O conteúdo desta pasta serve como apoio; a condução oficial da disciplina continua sendo a referência principal.
